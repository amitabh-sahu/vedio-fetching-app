# vedio-fetching-app
This is a vedio-fetching app build using create-react-app.

## Info
The app fetch the videos on every keystroke in the search bar. The socialboat-dev API is used to fetching videos. Page is fully responsive.

## Technologies
* React JS 17.0

## Setup
Open cli and use these commands:
```shell
git clone https://github.com/amitabh-sahu/vedio-fetching-app.git
cd vedio-fetching-app
npm install
npm start
```
